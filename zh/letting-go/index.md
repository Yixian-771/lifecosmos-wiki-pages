# 放下

**放下**，在生命禅院体系中，是迈向仙境的首要心法，也是修行修炼贯穿始终的实践要求。放下不是消极遗弃，而是卸除对过去功过、现有拥有、外部执念的一切抓持，使生命的灵体轻装前行。

> "若我要给你们讲佛法，首先的一条就是要大家'放下'。"
> （禅院文集·天启篇·放下）

放下的深度，决定一个人能到达的生命空间的高度。

## 核心要点

- **须放下什么**：理想、道德、真理、主义、家庭、民族、国家、宗教、政党、戒律；生死荣辱、善恶真假、功劳伟大；最终——放下自我（"我执"）
- **难度层次**：物质 → 亲情友情 → 僵化思想与我执（最难）
- **辩证核心**："放弃我，才能有我；执着于我，最终无我。"
- **与成仙的关系**：去千年界、万年界、仙岛群岛洲，放下是不可绕行的前提
- **第二家园**：放下一切后的现实支撑——"一无所有，拥有一切"
- **不强求**：不是机械戒条；"若放下折磨你，那就拿着，等自然成熟"
