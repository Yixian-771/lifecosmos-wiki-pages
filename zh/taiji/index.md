# 太极

**太极**，是生命禅院宇宙论中最核心的概念之一；上帝由无极中的偶然成序诞生，太极即是上帝诞生后宇宙的第一个存在形态，是宇宙从无序走向有序的转折点，是整个宇宙论体系的第一因。太极就是上帝，太极是道，是如来，是宇宙万物的源头与主宰。

> 无极生太极，太极就是上帝。太极自然而来，故自然就是上帝。
>
> ——《新时代人类八百理念》第503条
