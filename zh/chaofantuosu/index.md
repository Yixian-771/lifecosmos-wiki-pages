# 超凡脱俗

**超凡脱俗**，是修仙的必要状态，也是一品人生的核心标志。在生命禅院体系中，"俗"指名利权成败得失、机巧客套、谄媚摇尾，以一字蔽之：**躁**；"凡"指人情人缘人事人世，以一字蔽之：**闹**；仙境则用两字形容：**清静**。超凡脱俗，就是远离躁和闹，归于清静。其对立面是两种生命"疾病"：被权钱名色操控的"**鬼门关舞蹈症**"，以及颠三倒四、沾沾自喜的"**痴呆聪明症**"。摆脱这两症，便步入超凡脱俗的初级境界。

雪峰将超凡脱俗与"反常退进"并提——凡俗人喜欢的我不喜欢，凡俗人不喜欢的我偏喜欢，把凡俗人的价值观颠倒过来，就是神仙的价值观。
