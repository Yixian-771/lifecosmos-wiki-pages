# 地狱

**地狱**，在生命禅院体系中不是宗教传说或道德隐喻，而是宇宙36维空间中真实存在的生命下界，具体由三个平行世界构成：**阴间**、**冰冻层**和**火炼层**。地狱是灵性缺陷、自私和残忍生命在轮回中的必然归宿，也是上帝对背离上帝之道者公正惩罚的场所。

> 火炼层、冰冻层、阴间就是通常所谓的地狱的总称，地狱的生命一般不会直接进入人间。
>
> ——《新时代人类八百理念》第四版·第365条
