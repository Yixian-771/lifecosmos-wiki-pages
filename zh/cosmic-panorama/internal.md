# 宇宙全景图（内部版）

## 一、宇宙的本质与起源

**宇宙的起源**（《新时代人类八百理念》第402/416条）：

> 宇宙的起源不是宇宙大爆炸，而是**无极生太极，混沌变浑沌**。
> 宇宙诞生之前为"无极"——无外无内，无大无小，无边无际，无虚无实，没有时间，没有空间，没有物质，没有精神，是一派清明，又是一片混沌，是一切的有，又是一切的无。

**宇宙的本质与构成**（《新时代人类八百理念》第98/402/403/412/418条）：

- 宇宙是**上帝意识的产物**（理念402）
- 构成宇宙的**三要素**：**意识、结构、能量**（理念98/403）
- 宇宙中有**八种力**：磁力、引力、强力、弱力、构力、斥力、意力、灵力（理念403）
- 宇宙总能量为零；正负能量之和永远为零，由此产生**因果报应律**（理念412/418）
- **能量越大越无形，能量越小越成形**；振动频率越高越不可见（理念97/370）
- 宇宙全息：**其大无外，其小无内**，宇宙不会辨别人的善恶，只是如实反馈（理念413）

**大宇宙与地球宇宙**（《新时代人类八百理念》第415条）：

- 宇宙分大宇宙和小宇宙，人类所在的叫**地球宇宙**（小宇宙）（理念415）
- 地球宇宙是太极椭圆体结构，叫**法旋系**
- 借助空间隧道走遍地球宇宙，需约**1860年时间**（约5865.7亿光年）

**地球宇宙层级结构**：

| 层级 | 说明 |
|------|------|
| **法旋系（地球宇宙）** | 太极椭圆体，以天界为核心旋转 |
| **旋河系** | 法旋系中含3000个旋河系（大千世界） |
| **银河系** | 每个旋河系含3000个银河系（中千世界） |
| **太阳系** | 每个银河系含3000个太阳系（小千世界） |
| **地球** | 太阳系成员，**为地球人类专门设置并服务** |
| **月球** | 非自然生成，是天仙专门制造的仓库，存放飞碟 |

## 二、混沌与浑沌的区别

（《新时代人类八百理念》第417条，完整原文节选）：

> 混沌是一种**无序状态**，是宇宙未形成前的状态；
> 浑沌是一种**有序状态**，是宇宙形成后的状态。
> 混——混乱、无序；浑——浑然一体，完整不可分割；沌——漫无边际的汪洋状态。
> 混沌没有天地边界阴阳中道；浑沌有太极，是对立统一的那个"一"。

## 十三、核心金句

> 「宇宙为生命而存在，生命为宇宙而存在。生命亡，宇宙灭；宇宙死，生命绝。」（理念405）
> 「生命最佳的归宿是进入上帝的后花园——仙岛群岛洲生活。」（理念360）
> 「意识决定存在，我们就有36维空间。」（理念309）
> 「天堂属于开心快乐的人，不属于愁眉苦脸心事重重的人。」（理念485）
> 「有什么样的生命结构，就生活在什么样的空间。」（理念468）
> 「生命的本质就是灵魂，灵魂的本质就是意识，所以，意识即生命。」（理念376）
> 「死亡不是生命的终点站。」（理念308）
