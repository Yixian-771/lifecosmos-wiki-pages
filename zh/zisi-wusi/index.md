# 自私与无私

自私，是将个体利益置于一切之上的心性，是生命基因的癌细胞；无私，是超越自我局限的品质，是生命最完美的境界。天堂是无私者的生活疆域，地狱是极端自私者共同造就的苦海。无私者属天国生命，有私者属人间生命，私重者属地狱生命。
