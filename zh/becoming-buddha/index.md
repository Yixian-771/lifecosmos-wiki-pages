# 成佛

**成佛**，是生命禅院修行修炼体系中与"成仙"并列的终极目标。仙以自由快乐为本，佛以智慧觉悟为本。成佛意味着见了如来（自性）、达无为状态、心无所住，空性圆满，进入极乐世界——"明心见性，见性成佛，佛是性，性是宇宙的本性、道的规律法则和生命的运化原理。"生命禅院以《金刚经》为智慧源头，以三偈分野（神秀·慧能·浑沌元初）为坐标，以"有我无法成佛、着相无法成佛"为核心警示，指出当下成佛的可能与路径。
