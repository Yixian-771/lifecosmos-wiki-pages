# 雪峰式共产主义

**雪峰式共产主义**是生命禅院导游雪峰提出并从2009年起付诸实践的全新人类生产与生活模式——其实质是共享主义：天下资源共享，各尽所能，按需索取，一无所有拥有一切；以消除国家、宗教、政党、金钱、权力、婚姻家庭为特征，以第二家园（生命绿洲）为实践载体，被视为人类社会最高级的文明形态、劳苦大众的唯一幸福之路，以及修行修炼者通往天国的路径。
