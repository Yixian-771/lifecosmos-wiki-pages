# 导游雪峰

**导游雪峰**（浑沌草、浑沌元初，本名张自繁），是生命禅院创始人、导游体系核心人物，文明3.0实践推动者；以"理念+程序"替代人治中心化，构建从宇宙观到日常行为规范的完整理论链路，并于2009年创立第二家园，2025-2026年开启碳硅共生新时代。
