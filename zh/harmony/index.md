# 和

**和**，是生命禅院理论体系中"真、善、美、爱、信、诚"六大核心生命品质之外，同样在心灵花园建设中被明确列为重要花卉的品质之一，是第二家园宗旨的核心关键词，是自然之道的核心状态，是道的属性的外在体现，是人类通往天国的必要条件，也是文明3.0最直观的外在景象。和，不是妥协，不是软弱，不是回避矛盾，而是道在人间、人与人、人与社会、人与大自然中自然流淌的和谐状态——有道，自然有和；无道，争吵必随。

## 核心定义

1. **自然之道的核心是人与人、人与社会、人与大自然和谐共处。**（第713条）
2. **道的属性没有矛盾，有矛盾就不是道。人若有矛盾就没有入道，群体中有矛盾就证明这个群体没在道中。**（第444条）
3. **宁静和谐，自然柔和，和睦和气，风和日丽，和蔼和善，和谈和解，这是你的特质，也是生命禅院的期望。**（《雪峰文集·X03禅院篇·Phi-4——和舟草》）
4. **万物和谐，风调雨顺，人人开心、快乐、自由、幸福的生命禅院时代。**（第627条）
5. **和谐，就是要消除争吵、争夺、争抢、争战、争论、争执、争辩，不消除争，永远没有和谐，世界永远没有和平。**（《禅院文集·传道篇·与人争辩者没有善良之辈》）
