# 宇宙统一场

**宇宙统一场**，在生命禅院体系中指"适用于所有时间和所有空间中'放至四海而皆准'的定律"。科学家因只关注物质世界、只知四种力而始终无法建立宇宙统一场理论；雪峰指出宇宙中实有八种力（磁力、引力、强力、弱力，加上构力、斥力、意力、灵力），真正的宇宙统一场早已存在——即上帝的道，以宇宙三要素"意识、结构、能量"构成的浑沌体系。
