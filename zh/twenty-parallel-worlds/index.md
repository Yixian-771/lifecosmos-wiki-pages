# 20个集合体世界

宇宙中同时并存着20个世界（亦称"20个平行世界"），由导游雪峰依据立体坐标系的数学原理推导得出。我们人类所处的人间（XY世界）仅是其中之一，其余19个世界与人间同时存在，通过清凉界O点这一枢纽彼此相连——上帝的质即居于此处。理解20个集合体世界，是觉悟空间、走上帝之道的重要一步。
