# 麻将与浑沌文化

**麻将与浑沌文化**是生命禅院用以阐释中华文化之根——**浑沌文化**的核心论题。导游雪峰指出，中华文化的主要血脉是浑沌文化，而非儒释道；麻将演绎的正是浑沌原理：136张牌的天文数字排列组合，看似随机无序，实则有法有则，"手气"背后是宇宙浑沌机制在运行。理解麻将即理解浑沌，理解浑沌即理解上帝之道。
