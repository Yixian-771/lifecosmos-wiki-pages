# 成仙

**成仙**，是生命禅院修行修炼体系的核心目标之一，指通过净化心灵、了却尘缘、完善生命的反物质结构，从人升华为仙，进入千年界、万年界、极乐界等高层生命空间，告别轮回，享受永恒的自由快乐。正如导游雪峰所言："人终其一生的努力，应当放在成仙之上——不成仙，就是囚犯。"
