# 极乐界天仙

在生命禅院体系中，**极乐界天仙**是仙品五级中的最高品——天仙——的居所与生命形态，也是人类修行修炼所能企及的最高境界；天仙的大本营是极乐界的仙岛群岛洲，那里有800亿个大小与地球相当的岛屿，每岛居住一位天仙，天仙是全宇宙最自由最快乐最幸福的生命，不承担任何责任，不尽任何义务，以自娱自乐为本，拥有64种神通，以幻化自在为日常。
