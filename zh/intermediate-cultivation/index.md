# 中级修行

**中级修行**是生命禅院修行修炼体系的第二阶段，在初级修行的基础上，通过崇拜上帝、悟道、识佛，创建崭新的生命轨道，进而入俗脱俗、涉世出世、就凡超凡，最终达到"物我两忘，靠心像生活"的境界。中级修行共11个方面，配有专属的中级祈祷语。其最高层次"物我两忘"既是中级修行的终点，也是高级修行的起点。
