# 红尘

红尘，在生命禅院体系中，是世俗物质世界的总称，其实质是三十六道八卦阵的全体。以"情阵"为核心，红尘是轮回、自私、业力与烦恼的综合场域；既指外部世俗环境，更指内心的执着与牵挂状态。看破红尘、超越红尘是修行修炼的重要阶段；彻底脱离红尘，是成仙成佛、回归生命本源的必经之路。

> "什么是红尘？红尘就是三十六道八卦阵，我们常说的'身在红尘中'，实际上就是身在三十六道八卦阵中。"
>
> *（雪峰文集·警世篇·早日看破红尘）*
