# 觉悟

**觉悟**，是生命禅院体系修行修炼的核心目标——通过六根（眼耳鼻舌身灵）对外界刺激的感受与深度思索，从迷惑走向明白，从蒙昧走向清明，从死亡走向永生；觉悟的极至就是佛，一切修炼的目的就是为了觉悟。
