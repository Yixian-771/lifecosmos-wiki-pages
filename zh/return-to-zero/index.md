# 归零

**归零**，是生命禅院修行修炼体系中最核心的意识境界概念之一——将自我偏见、执见全部放弃，将固有意识全部清空，使自己一无所有，将自己置于已经死亡的境地；归零了，就是佛了；归于零，就拥有一切；上帝是零，归零，就与上帝一体，上帝永恒，我也永恒。归零是无相思维的核心实践，是八大思维阶梯第七阶的本质操作，是修行通往极乐世界的必经关口，也是"一无所有拥有一切"这一禅院生活原则的意识基础。
