# 灵性

**灵性**，是生命禅院理论体系中的核心中轴概念之一，既是区分不同生命层次的重要标志，也是修行修炼中"意识升华—结构优化—生命去向"路径的关键变量；灵性是生命在与"灵"的连接中呈现出的高层次觉知能力、活力状态与顺道而行的综合品质。
