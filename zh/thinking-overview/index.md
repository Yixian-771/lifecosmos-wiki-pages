# 思维（总论）

> 思维是意识的核心成分，思维能改变意识。要想改变自己的生活方式，要想进入一个理想的家园，首先要改变思维模式和方式。有什么样的思维方式，就会有什么样的生命存在形式。
>
> ——《建设我们理想的家园》，雪峰文集

**思维**是思想、思索、思考的维度与过程，是意识的核心组成部分，也是决定生命层次与去向的关键力量。禅院体系将思维分为八大阶梯，从物质思维到浑沌思维依次递进；思维层次越高，生命境界越深，越接近上帝。改变思维，是改变生命去向、进入理想家园的根本之道。
