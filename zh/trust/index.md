# 信任

**信任**，在生命禅院体系中，是"毫不怀疑地相信"——是一种巨大无形的能量，是天堂生命的品质与专利，是人与人之间最珍贵的资源。信任创造天国，不信任则使天堂变成地狱。生命禅院以"信"为七大品质之一，视"大信"（对佛祖上帝、对导游的百分之百信任）为文化建设的首要基石。

> "信任是蜂蜜，不信任是一把刀。"
>
> ——雪峰《新时代人类八百理念》第137条
