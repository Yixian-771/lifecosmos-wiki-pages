# 上帝

**上帝**，在生命禅院理论体系中，是宇宙的创造者与主宰，是太极（宇宙从无极中诞生的第一个有序结构），是道、如来、耶和华、真主安拉的同一本体，是真善美爱信诚的总体属性，是禅院草修行修炼的最高敬畏对象。
