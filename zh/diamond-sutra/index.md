# 《金刚经》

**《金刚经》**，全称《金刚般若波罗蜜经》，是释迦牟尼佛专为"发大乘、发最上乘者"所说的至高佛法，是一切佛经的源头。在生命禅院体系中，《金刚经》被誉为"智慧的喜马拉雅山"——其核心教义无相（凡所有相，皆是虚妄）、无我（四相皆空）、心无所住（应无所住而生其心）与生命禅院的无相思维·浑沌思维高度契合。雪峰以浑沌元初之法名接续佛法衣钵，著成禅院文集《成佛篇》24篇，全面解读《金刚经》，指出生命禅院即《金刚经》的后续内容。
