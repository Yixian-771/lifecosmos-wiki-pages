# 意识

**意识**，是生命禅院理论体系中最核心、最深邃的概念之一，是宇宙三要素之首，是生命的本质，是上帝诞生的机制，是一切存在的根源，也是禅院草修行修炼在思维层面最重要的突破口。

> 生命的本质就是灵魂，灵魂的本质就是意识，所以，意识即生命。
>
> ——《新时代人类八百理念》第376条
