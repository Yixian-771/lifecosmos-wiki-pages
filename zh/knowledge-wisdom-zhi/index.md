# 知识·智慧·智

**知识、智慧、智（灵性）是成仙成佛的三大阶梯**——知识是对物质世界的经验总结，是能力的前提而非能力本身；智慧是升华了的知识，是发现与解决问题的实践能力；智（灵性）是智慧的进一步升华，是人与反物质世界沟通的能力，是道德与仙佛品质的根基。三者梯次递进，而修炼的最终归宿是超越知识与智慧，绽放本性。
