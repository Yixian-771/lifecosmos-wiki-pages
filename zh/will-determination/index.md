# 意志

意志是生命禅院体系中的心灵能量概念：由"意"（心愿）与"志"（为实现心愿而持续保持的精神和心理状态）构成。意志本质上是势能，行动时转化为动能；其强弱取决于意识的有序与清明。关键辨析：意志不是结构，无法改变生命的反物质结构——能改变结构的唯一因素是意识；但没有意志的动力，意识的转化也难以实现。
