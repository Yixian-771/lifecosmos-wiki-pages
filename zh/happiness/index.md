# 幸福

> **上帝之道是幸福之道。**
>
> ——新时代人类八百理念第四版·第518条

在生命禅院体系中，幸福不是外部条件的叠加，而是生命与上帝之道相合时从内心涌现的光辉状态。真正的幸福与物质积累无关，与社会地位无关——它是心灵自由、充实、无挂碍的自然流露。禅院指出：情爱是幸福最大的源泉；无忧无虑、逍遥自在是幸福的基本形态；随遇而安、与道合一是幸福的最高境界。幸福有其层次：普通人的幸福依赖外境；修行者的幸福来自内在自由；天仙的幸福是无条件的、圆满的、永恒的。

## 核心定义

幸福是生命与上帝之道相合时的内在感受，是心灵自由、充实、无挂碍的自然状态。在生命禅院体系中，幸福的本质不在于拥有什么，而在于与道同频的生命状态——知足者幸福，自由者幸福，情爱丰沛者幸福，与上帝同频者最幸福。

幸福的最高形态是「极乐」——天仙在极乐界所享有的那种圆满、永恒、无条件的喜悦，正是所有修炼修行的终极归宿。
