# 仙性

**仙性**，是仙的生命结构所具有的特性。在生命禅院体系中，"性"是结构的特征，有什么样的结构就有什么样的性；人要成仙，根本路径不是苦修戒律，而是"随仙性而行"——让意识和行为逐渐向仙的结构靠拢，久而久之，生命的反物质结构自然转化，成仙可达。雪峰将仙性凝结为**八大特征**，并将之与灵性等同："灵性就是仙性，以后的一千年，人类中的所有成员会像仙那样生活。"
