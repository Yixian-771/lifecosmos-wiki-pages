# 高级修行

**高级修行**是生命禅院修行修炼体系的第三阶段，以物我两忘为起点，核心在于冲出一切规章制度与戒律形式、悟佛性、创造天仙生命结构，最终破茧成仙、进入负宇宙高层生命空间。正如导游雪峰所言："初级修行是作茧，中级修行是为了在茧中孕育新生命，高级修行就是破茧而出羽化成仙。"
