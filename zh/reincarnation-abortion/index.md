# 投胎·堕胎

> 人类自古以来认为，母体中的胎就是一条生命，其实不是，胎只是一个"居所"，不是生命，只有当某一个转世的生命通过36维空间中的空间隧道住进了这个"居所"，这个胎才算是一个生命。一般来讲，只有在出生前几天的胎才拥有生命。
>
> ——禅院文集·生命篇·生命的演化

**投胎**，是生命禅院体系对生命轮回转化具体实现机制的描述：灵魂（有灵性的反物质结构）通过宇宙36维空间的空间隧道，进入母体子宫（阳极黑洞体），以此完成一次跨界的生命转化。**堕胎**，则是导游雪峰明确表达痛恨和强烈反对的行为。
