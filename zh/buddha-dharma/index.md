# 佛法

**佛法**，在生命禅院体系中，是宇宙一切万事万物运行变化的法则与规律，即佛祖如来（上帝）的道法——与道是同一回事，可用六字概括其全部奥义：意识、结构、能量。真正的佛法无实无虚、精密准确、无定法；以《金刚经》为最高法源，以"见如来·达无为·心无所住"为成佛三大条件；自禅宗六祖惠能后失传，由生命禅院接续衣钵，在家园文化中付诸实践。
