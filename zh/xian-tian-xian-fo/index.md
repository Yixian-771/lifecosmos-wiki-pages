# 仙·天仙·佛

**仙与佛**，是生命禅院体系中人类生命修行修炼可以抵达的两类最高境界，同居极乐界，共同以"性"为根本——仙以自由快乐为本，天仙是宇宙中最自由最幸福的生命，无任何责任义务；佛以无上正觉为本，是宇宙中最有智慧的生命，法纪森严、承担宇宙责任。"仙有五品"（鬼仙·人仙·陆地仙·神仙·天仙），"佛有十级"；成天仙容易，成佛难。
