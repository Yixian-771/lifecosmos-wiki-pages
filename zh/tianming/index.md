# 天命

**天命在生命禅院体系中涵盖三个层次**——其一，上天赋予每个生命与生俱来的才能与使命；其二，道（程序）预先编排的人生轨迹与宿命；其三，宇宙万物运行的必然法则。"知天命"是修炼的重要境界：认清自己是什么材料，做自己，顺天应命。而"超越宿命"则是更高追求：98%的人遵循宿命程序，少数禅院草通过反常思维、开启灵觉，从必然王国走向自由王国。
