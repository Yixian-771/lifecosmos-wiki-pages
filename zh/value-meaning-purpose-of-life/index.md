# 人生的价值·意义和目的

**价值、意义、目的三词各有所指**——人生的价值在于能否使生命延伸到高层生命空间；人生的意义是创造财富和欢乐、取悦上帝、求道成仙；人生的目的是消业转化，同时活出开心、快乐、自由、幸福的当下。人生是生命长河中短暂的一段旅行，充分利用这段旅行的价值，便可"化腐朽为神奇，化死亡为永生"。
