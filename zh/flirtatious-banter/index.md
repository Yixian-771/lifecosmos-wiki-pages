# 打情骂俏

打情骂俏，是生命禅院"浑沌修炼法"中的重要环节，指家园兄弟姐妹之间随缘、无固定对象、游戏性的情趣调逗与亲密玩笑。其目的不是占有某人，而是通过这种过程打开心量、消融忌妒与占有心，将人的意识升华为仙的意识，为进入高级修炼奠定基础。
