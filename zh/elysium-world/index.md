# 极乐界

**极乐界**，是天国三界之一，与千年界、万年界共同构成天堂（天国）整体结构，被定位为三界中层次最高、范围最广、终极归宿指向最明确的高层生命空间，属负宇宙范畴，与仙岛群岛洲密切关联。
