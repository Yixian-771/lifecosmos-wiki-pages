# 精神（总论）

**精神**，在生命禅院体系中，是"精"与"神"的综合体，是生命体的支柱，是物质与反物质相互联系、互相作用的桥梁与结合部。精神连通物质与反物质两界，是生命从物质世界向心灵世界升华的中间地带。人无精神必亡；人有精神可以镇百邪，不仅疾病不染，还能吸收天地之灵气。
