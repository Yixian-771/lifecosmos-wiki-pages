# 宇宙（总论）

**宇宙**，在生命禅院体系中，是"由无限空间和无限时间构成的物质和反物质的综合体"。宇宙的本质是浑沌，构成要素是意识、结构、能量，起源于无极生太极、混沌变浑沌，运行的核心血液是道，创造者与主宰者是上帝。宇宙与生命互为存在，宇宙为生命而存在。认识宇宙，是认识生命、认识道、认识上帝的前提。
