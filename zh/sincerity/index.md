# 诚

**诚**，是生命禅院理论体系中"真善美爱信诚"六大核心生命品质之一，是上帝的属性，是做人的根本基础，是心灵花园里最重要的花卉之一，也是前往天国千年界的必要条件；涵盖诚实（内心与外在言行的完全一致）与守信（承诺与行动的完全一致）两大维度。

> 诚实，是做人的上上之策。
>
> ——《新时代人类八百理念》第116条
