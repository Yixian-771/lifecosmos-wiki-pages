# 祈祷

**祈祷**，是人与上帝（道）建立直接联系的核心方式之一，与冥想并列。在生命禅院体系中，祈祷不是宗教仪式，而是心灵与上帝之间真实的沟通——人的内心思维化为思维波向宇宙发出，道具有瞬时感应特性，上帝必然回应。祈祷是修炼体系中不可缺失的日常意识活动；没有祈祷，就得不到神助。

> 祈祷具有不可思议的法力。当自己被环境束缚而实现不了美好理想时多祈祷吧！
