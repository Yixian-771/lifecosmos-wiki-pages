# 418

**418（4月18日）**，是生命禅院**第二家园的诞生日**。2009年4月18日，导游雪峰带领第一批禅院草，在中国大地上正式开启了人类新生活模式——第二家园——的实践之路，此后每年的418成为全体禅院草共同铭记的神圣日期。
