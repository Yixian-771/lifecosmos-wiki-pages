# 少私寡欲·宠辱不惊

**少私寡欲·宠辱不惊**，是生命禅院体系中修仙起步阶段的两项核心品质——少私寡欲意味着减少私心、降低物欲，不让功名利禄成为修仙的陷阱；宠辱不惊意味着无论得宠还是受辱，内心保持如如不动，不做外境的奴隶，守住天性的完整。导游雪峰指出，修仙者的注意力应从魄（肉体利益）转向魂（精神心灵），只有如此，方能踏上真正的修仙路径。

> 功名利禄、荣华富贵是修仙的陷阱；荣耀耻辱、宠幸落贱是修仙的障碍。只有少私寡欲、宠辱不惊，才能踏上修仙的路径。
>
> ——禅院文集·修仙篇·少私寡欲、宠辱不惊
