# 仙子·仙女

在生命禅院体系中，**仙子**是对禅院男性成员的称谓，**仙女**是对禅院女性成员的称谓。这一命名源自宇宙论框架：千年界、万年界等高层生命空间中，男性生命称仙子，女性生命称仙女。禅院草以此互称，彼此以高层生命标准期许。禅院的核心理念"乾坤颠倒，仙女为尊"——仙子为绿叶，仙女为花蕊；危险面前仙子在前，享受面前仙女优先。
