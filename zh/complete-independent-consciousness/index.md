# 完整独立意识

完整独立意识，是生命禅院修行体系的核心概念之一：一个人的人生观、价值观、生命观、宇宙观符合上帝之道，在此基础上能够游刃有余地处理人生旅途上的一切事务，而不受他人意识的左右和影响。它是达到自洽——修炼最高境界——的前提，也是禅院草从必然王国进入自由王国的关键。
