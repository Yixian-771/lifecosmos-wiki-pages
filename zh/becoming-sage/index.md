# 成圣

**成圣**，是生命禅院修行修炼体系中最高的人间境界，指一个人彻底化去自我、以利益天下苍生为唯一驱动，达到无我、无私、无执、无恨的生命状态，成为真正意义上的圣人，并由此开启进入极乐界仙岛群岛洲的天命之路。
