# 四随（随遇而安·随缘而化·随性而动·随机而作）

**四随**是生命禅院体系中成仙修炼的核心行为法则：随遇而安、随缘而化、随性而动、随机而作——安于任何遭遇处境而心不怨恨，顺缘而动不强求，依天性行事顺道而行，把握时机全力以赴；能做到四随并无我无私无执者，便是仙，所呈现的状态便是仙态。
