# 宇宙起源

**宇宙起源**，是生命禅院宇宙论中的根基词条，回答"宇宙从何而来、如何展开、为何有序运行"这三个核心问题；该体系以"无极生太极，混沌变浑沌"为总纲，提出"结构产生意识，上帝由此诞生，宇宙由此展开"的完整链路。

> 宇宙是上帝意识的产物；宇宙起源不是大爆炸，而是无极生太极、混沌变浑沌。
