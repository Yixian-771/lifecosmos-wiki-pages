# 无极

**无极**，是生命禅院宇宙论中的起点概念，用于回答"宇宙诞生前是什么状态"这一根本问题；无极不是绝对虚无，而是"无时空、无形态、无有序结构，但具有潜在能量运动"的前宇宙状态，无极之后是太极，上帝由"无序能量偶然成序、结构产生意识"的机制中诞生。
