# 真

**真**，是生命禅院理论体系中"真善美爱信诚"六大核心生命品质之首，是宇宙的本体，是上帝的属性，是修行修炼的根本方向，是心灵花园最核心的种子。

> 宇宙中只有一个真，除了上帝之外，其余的都是幻化的，都是上帝的艺术表现形式。
>
> ——导游雪峰《雪峰文集·心灵篇·真善美——人性的核心价值》
