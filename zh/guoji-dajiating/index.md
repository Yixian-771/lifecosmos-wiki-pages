# 国际大家庭

国际大家庭，是生命禅院体系对未来人类文明形态的核心构想。它超越国家、民族、宗教、政党的一切边界，以"全世界各民族、各宗教、各政党、各团体的成员都是国际大家庭不可缺少的一份子"为基本原则，以第二家园（生命绿洲）为实践载体，以有福同享、有难同当、资源共享为生活模式，以心灵净化为根本途径。在新时代人类八百理念中，"万法归宗，万教归一，天下一家，世界大同"被列为新时代的主旋律。终极愿景是在地球上创建256处家园，使人类告别战乱、饥饿与冲突，最终将地球变为人间天堂，成员亦将在离开人间后被接迎至天国各界。
