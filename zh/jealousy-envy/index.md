# 忌妒·嫉妒

忌妒（嫉妒）是比较之心的直接产物，是心灵丑陋的写照，也是生命无法升华、无缘天国的核心障碍——从情感忌妒到强烈嫉恨，它固化格局、消耗能量，与仙的境界水火不容。
