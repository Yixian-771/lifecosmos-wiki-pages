# 傲慢

傲慢，是无知在言行中的必然显现，是生命与魔性同频的第一步，也是成仙成佛路上最难察觉、破坏力最大的心灵杂草——宇宙的天使长因傲慢成了魔王，人间的修行者因傲慢而前功尽弃。
