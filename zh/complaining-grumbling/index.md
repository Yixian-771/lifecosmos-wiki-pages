# 埋怨·抱怨

埋怨与抱怨，在生命禅院体系中被视为导致人生痛苦的十八因素之一，是"射向他人的毒箭"，是进入天国前须洗涤的二十项心灵尘垢之一。其根源在于：将自身处境的不如意归咎于外部世界，而忽视了"现实是自己意识的投射，一切皆因果"这一基本认知。

生命禅院的修炼要求是：不怨天、不怨地、不怨社会、不怨他人，将向外的指责转化为向内的反省与感恩，因为越是自私的人，埋怨越多；而有感恩心的人，没有怨气。
