# 忏悔

**忏悔**，在生命禅院体系中，是人与创造主——上帝之间的私事，是人认识到自己言行偏离上帝意愿后，用心灵诚恳向上帝诉说过错、祈求宽恕的行为。忏悔不是宗教仪式，不对人，只对上帝，以心灵为唯一媒介。忏悔是净化心灵的有效法门，是消除孽障的一剂良方，是摆脱困境的第一法门，也是升华生命品质的必要条件。

> 忏悔是生命品质升华的必要条件，无忏悔，无升华。
