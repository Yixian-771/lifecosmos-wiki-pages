# 缘

**缘**，是生命与生命之间、生命与万事万物之间所结下的无形引力与债务关系。一切相遇、悲欢、离合、苦乐，皆有缘之所驱。缘是反物质存在，无形无相，却有影有踪；缘既是人生轨迹的编写者，也是生命轮回的根本原因。了尘缘、结仙缘，是修行修炼从人到仙的核心课程。
