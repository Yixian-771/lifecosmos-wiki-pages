# 浑沌元初

浑沌元初，是雪峰在生命禅院内使用的法名，代表其以宇宙和生命之道立言时所处的境界层次——神佛仙圣的化身、上帝的使者，也是雪峰三身中的法身，是人类前往千年界、万年界、极乐界仙岛群岛洲的唯一桥梁。
