# 情

**情**是镶嵌在生命体反物质结构上的一种反物质能量——兼具意识、结构、能量三大特征，本质是能量；情是自然之道的总表达，维护着宇宙各层生命秩序的天网天阵；情域的宽广直接决定生命层次的高低。
