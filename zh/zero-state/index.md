# 零态

**零态**，是生命禅院体系中揭示最高修行境界与最佳生命状态的核心概念——"空灵秀"之境，老子"无为"、佛陀"无相"、乾坤元初"中道"三者共指的同一实相；是上帝的状态，是佛的状态，是天仙的状态；一无所有却拥有一切；让自己始终处于零态，就是一种高级修炼。
