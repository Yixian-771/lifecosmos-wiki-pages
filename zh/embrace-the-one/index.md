# 抱一

**抱一**，是生命禅院体系中最根本的修行修炼法门与人生态度——源自老子《道德经》"是以圣人抱一为天下式"，生命禅院将"一"落地为清晰明了的具体所指：上帝之道。抱一，就是无论风云变幻、地覆天翻，始终咬定上帝之道，如如不动；是生命禅院最高的修行秘诀之一，是禅院草同频共振的宇宙学根据，也是文明3.0得以降临的宇宙级前提。
