# 上善若水

**上善若水**是生命禅院体系对老子《道德经》核心命题的阐发。雪峰指出：水"几于道"，以七种品性——滋润万物而不争、洗涤尘垢而不居高处、随形而形、幻化气云无处不安、柔弱谦卑、能流能停、能歌能寂——承载了道的全部品质。欲善者学水，求道者学水；人能若水，则与道同体，享万古长流。
