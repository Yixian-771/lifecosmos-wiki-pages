# 高级修炼

**高级修炼**是生命禅院三级修炼体系的最高阶段，目的是"炼性"——涤去人性中的杂质，恢复上帝赋予生命的本来特性，进而创造天仙的生命结构，进入高层生命空间。其核心法门涵盖十讲：在爱里升华、神通变化、闭眼看世界、隐遁、畅游思维怪圈、玩五行游戏、常处零态、把自己表达清楚、创造天仙结构、与上帝对话交流。进入高级修炼，即等于开始探索反物质世界的生活。
