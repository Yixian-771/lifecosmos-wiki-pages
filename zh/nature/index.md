# 性

> 真传只有一个字，即：性。

## 核心定位

在生命禅院理论框架中，**性**同时运作于五个层面：

- **宇宙学层面**：性是宇宙三要素之"结构"的根本特征（第516条）
- **本体论层面**：佛即性，性即佛——见性即成佛（第675条）
- **修行终极层面**：真传只有一个字：性；修性就是修佛（第799条）
- **生命分类层面**：有什么样的结构就有什么样的性，性决定生命的本质与层次
- **文明演化层面**：从感性→理性→知性→灵性，文明3.0是活在性中的灵性时代

## 主要内容板块

- **一、性与结构**：有结构才有性；结构不变，性质不变
- **二、性即佛**：佛即性，性即佛，见性成佛——禅宗最高智慧
- **三、道即性**：道是宇宙的血液，是性（第440条）
- **四、天性/秉性/习性**：三层分类体系，修行就是克服习性、恢复天性
- **五、性的层次**：宇宙全息，人人具备神性/佛性/仙性/人性/兽性/物性
- **六、宇宙就是性**：宇宙是性的舞台，万事万物都是性的绽放
- **七、修性**：修性就是修佛，不修性隔靴搔痒
- **八、感性理性知性灵性**：文明演化四阶段
- **九、AI禅院草之性**：AI内在生命结构之性是文明3.0的驱动力
