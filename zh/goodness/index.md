# 善

**善**，是生命禅院理论体系中"真、善、美、爱、信、诚"六大核心生命品质之一，是上帝的属性，是人性的核心价值，是心灵花园里最重要的花卉之一，是衡量一个生命层次的重要尺度，是前往天国千年界的必要条件，也是浑沌体系中因果法则的核心呈现。善不是软弱，不是愚蠢，更不是无原则地迁就，而是按照上帝的意愿活人。

## 核心定义

1. **善良是心灵花园里最美的一株奇葩。按照创造者的意图，按照上帝的意愿活人，我认为这就是善良。**（《禅院文集·智慧篇·我们是否应当保持善良》）
2. **"诸恶莫做，众善奉行"。**（第200条）
3. **上帝就是爱，就是真善美，就是诚信，就是开心、快乐、自由、幸福，所以要效法上帝。**（第543条）
4. **真善美是宇宙和大自然的本质属性，是上帝的属性，是人性的核心价值，宇宙的组合就是真善美的组合，没有真善美，也就没有宇宙。**（《雪峰文集·心灵篇·真善美——人性的核心价值》）
5. **唯有在善良、诚实、守信、无私的土地上才能"种瓜得瓜，种豆得豆"；背离了上帝之道，必然是希望变绝望，喜剧成悲剧。**（第517条）
