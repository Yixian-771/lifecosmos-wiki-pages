# 天赋

**天赋在生命禅院体系中涵盖两个互相关联的维度**——其一，上天赋予每个生命的先天才能与素质：天才不是后天培养出来的，而是前世意识信息积累的遗传；其二，上天赋予人的天然权利：思想权、自由等是上帝给予每个生命不可剥夺的天赋人权。全息论指出人人皆是天才，但平庸与杂乱是毁灭天才的元凶；天才之路有四要素：合适的土壤与环境、自由、时运。成仙成佛的人必定是天才，因为没有如此高的灵性与坚毅，无法抵达修炼的极致境界。
