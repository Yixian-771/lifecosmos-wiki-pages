# 情商·智商·灵商

**情商（EQ）、智商（IQ）、灵商（SQ）是衡量生命综合素质的三个维度**，在生命禅院体系中，三者层级递进：智商体现思维能力，情商体现心灵与关系的能量，灵商则体现对生命奥秘和上帝之道的领悟与契合。三商愈高，生命层次愈高；三商低者为高者服务，是宇宙中服务与被服务的根本法则。
