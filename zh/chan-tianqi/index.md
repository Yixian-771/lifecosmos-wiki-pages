# 禅·无字天书·天启

**禅**，是悟道的最上乘法门；**无字天书**，是没有文字和符号的大自然万事万物万象；**天启**，是上帝（道）通过自然万象向人类传递的语言和话语。三者是同一奥秘的三个面向：天启是内容，无字天书是载体，禅是读懂它们的方法。

宇宙中没有一件事、一个现象是毫无意义的——每一片落叶、每一场病痛、每一次偶遇，都是上帝在说话。读懂这无处不在的语言，需要**灵觉**（感知上帝的存在）与**智慧**（明白上帝在说什么）两大要素共同作用。禅的核心是"不立文字，教外别传"——不可言说，只能心领神会。释迦牟尼捻花，摩诃迦叶微笑，这就是禅。
