# 无神论者

无神论者，在生命禅院体系中，是指否定上帝存在、不相信生命不灭与因果轮回的人。其思想建立在宇宙大爆炸、生命进化论、"人死如灯灭"三大基石之上——导游雪峰逐一驳斥了这三大基石，并以"给无神论者的八点忠告"系统阐明：无神论者缺乏道德根基、长远理想与天网约束，无法成为禅院草，也无缘天国。
