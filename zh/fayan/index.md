# 法眼

**法眼**是佛陀所传的**五眼体系**中第四层次的智慧之眼，位于慧眼之上、佛眼之下，是洞见宇宙万法运行规律、鉴别正邪虚实、透达生命奥秘的能力。

生命禅院依据佛教五眼体系，将智慧分为五个层次（**八百理念#88**）：

> **肉眼智慧 → 天眼智慧 → 慧眼智慧 → 法眼智慧 → 佛眼智慧**

雪峰在《我为大家开法眼》（2007年）中指出：

> 具备法眼后可以**智慧顿出**，"一览众山小"，可以**鉴定正邪**，可以**明白生命奥秘和天国情景**。

**开启法眼的钥匙**，是紧紧把握宇宙三要素——**意识、结构、能量**。一旦弄清楚三者之间的关系，法眼就开了，再不会被任何高深的说教或理论迷惑。

法眼一开，便站在了人类历史上所有智慧最高者的行列；再加上敬畏上帝、心怀天下、为苦难者谋幸福，就接近神了。
