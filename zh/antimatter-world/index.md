# 反物质世界

**反物质世界**，是生命禅院宇宙论体系中与物质世界并列的另一半宇宙实相，是生命灵体真实存在和活动的空间，是上帝、神佛、天仙生命形态的本土，也是禅院草修行修炼最终要进入的高层生命空间。在生命禅院体系中，反物质世界等同于"负宇宙"，是物质世界的本源。
