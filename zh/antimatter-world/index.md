# 反物质世界

【概念入口页】

反物质世界，是生命禅院宇宙论体系中与物质世界并列的另一半宇宙实相，是生命的灵体真实存在和活动的空间，是上帝、神佛、天仙生命形态的本土，也是禅院草修行修炼最终要进入的高层生命空间。

在生命禅院体系中，反物质世界等同于"负宇宙"——物质世界是"正宇宙"，反物质世界是"负宇宙"。负宇宙是物质的本源，物质世界是反物质世界的反映与表象。

反物质世界并非科学意义上的"反粒子构成的物质"，而是指一切没有质量、没有重量、肉眼看不到、任何仪器探测不到、但意识能够观察体验的领域——包括梦境、时间隧道、空间隧道、极乐界十大部洲、阴间，以及意识、灵魂、心灵感应的通道等。
