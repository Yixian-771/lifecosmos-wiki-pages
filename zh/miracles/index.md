# 奇迹

奇迹，在生命禅院体系中，是指"认为不可能发生的事发生了，或最美好的愿望实现了"。奇迹并非神秘的偶然恩赐，而是宇宙程序在条件具备时自动运行的结果；通过专注与坚持、心灵净化与信仰，人人皆可触动程序，让奇迹在不知不觉中自然降临。

> "追求奇迹的永远得不到奇迹，坚守平常的最终将有奇迹降临。"
>
> *（禅院文集·传道篇·不要祈祷奇迹的发生）*
