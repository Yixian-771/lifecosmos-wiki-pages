# 天性

**天性，即如来本性，是创造者赋予每种事物和生命的本质属性与原始特性**——先天澄澈清明、一尘不染，体现上帝神佛的特征。在生命禅院体系中，天性属全属（全体共有），有别于属类属的秉性和属个属的习性；彰显天性、随天性而行，是上帝之道八大内涵之一；仙佛即不受家庭、国家、文化等因素制约而随天性而行的生命。
