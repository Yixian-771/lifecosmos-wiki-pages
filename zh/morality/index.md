# 道德

**道德**，在生命禅院体系中有其根本定义：维护如来本性（天性）的行为方式就是道德，违背如来本性的行为方式就是不道德。道德的唯一尺度，是是否有助于人获得开心、快乐、自由、幸福——这一定义超越一切宗教戒律与人间法规，直接对接宇宙因果律。
