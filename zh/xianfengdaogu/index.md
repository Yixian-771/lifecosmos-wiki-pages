# 仙风道骨

**仙风道骨**，是成仙者在气质、风度与内在品质上的综合展现。"**仙风**"指仙人的风度、风采、风姿、风情、风范；"**道骨**"指得道者骨子里所展现出的气概、气度、胆略、魅力和清雅。道骨是内涵，仙风是外表，"内涵和外延缺一不可"。在生命禅院体系中，仙风道骨是**前往千年界、万年界、极乐界仙岛群岛洲的必要条件**之一，也是修行修炼达到高级阶段后自然显现的生命状态。雪峰将其凝结为**八大内涵素养**和**八大外表素养**，并以64个细节加以落实。
