# 初级修行

**初级修行**是生命禅院修行修炼体系的第一级，以51条日常行为规范为核心，以向真主-佛祖上帝祈祷为起点。其目标是清理负面思路、规范日常言行，帮助修行者在凡俗社会中走向健康、富裕、自由之路，并为中级修行打下坚实基础。导游以"作茧"比喻初级修行——规范之茧保护生命的孕育，是通往中高级修行乃至成仙的必经阶梯。
