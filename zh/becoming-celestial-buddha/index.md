# 成仙成佛

**成仙成佛**，是生命禅院体系修行修炼的终极目标，也是"生命的最高境界"——通过净化心灵、完善生命的反物质结构、觉悟生命宇宙的真相，从人升华为仙或佛，进入天国高层生命空间，成就无上正觉。仙以自由快乐为本，佛以智慧觉悟为本，二者殊途同归，皆是人所能抵达的最高生命归宿；而其根本前提，是"把人性完美了，仙佛自然而成"。
