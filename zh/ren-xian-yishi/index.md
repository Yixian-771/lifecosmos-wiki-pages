# 人的意识·仙的意识

人的意识，是束缚生命于人类层次的心理结构与思维定势，涵盖国家、民族、家庭、宗教等集体认同，以及忌妒、贪婪、恐惧、占有等情绪与欲望模式；仙的意识，是超越人类局限的生命意识，以随缘放旷、任运逍遥、把生命交给上帝为核心。在生命禅院体系中，生命的去向由其反物质结构决定，而反物质结构由意识决定——格式化掉人的意识、安装仙的意识，是生命从人界升入仙界的唯一路径。
