# 颠倒梦想

**颠倒梦想**，出自《般若波罗密多心经》"远离颠倒梦想，究竟涅槃"，在生命禅院体系中指一切因心有执着、挂碍、比较、恐惧而生的错误认知与妄想——患得患失、辗转反侧、执相不放、贪生怕死，皆是颠倒梦想；心无挂碍则无颠倒梦想，可究竟涅槃，前往天国仙界。
