# 大乘心愿

大乘心愿，是放弃自己、全身心投入普度众生事业中的心愿，是超越个人解脱、以天下苍生为念的宏大志愿；最上乘心愿，是走上帝如来佛祖的道，让众生达至无相思维境界、进入浑沌思维状态的心愿。在生命禅院体系中，大乘心愿是禅院草的基本资格，是获得无上正觉智慧的前提，也是成就他人与成就自己统一的根本路径。
