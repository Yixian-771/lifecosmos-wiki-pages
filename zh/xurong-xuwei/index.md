# 虚荣与虚伪

虚荣，是以他人的评价和赞美为满足感来源的心性，是人性的癌细胞；虚伪，是心里想要一套、言论行动上却表现截然相反的状态，是生命结构不完美的症状之一。两者互为表里。在生命禅院体系中，真实（纯朴、顺性、自然）是与虚伪对立的正面品质，也是进入天国的必要条件。
