# 明心见性

明心见性是生命禅院体系中成佛的核心路径：明白"心"（眼耳鼻舌身意对环境感应所生起的念头，因缘而生）的本质，进而见到"性"（如来、浑沌元初、宇宙本性、上帝意识）。心无所住，性自现；见性即成佛。
