# 无极体制·太极体制

**无极体制与太极体制**，是生命禅院新时代社会蓝图的核心方法论。无极体制对应宇宙的整体感应特性，落实为信息公开透明、良知良能可获真相的社会机制；太极体制对应宇宙的有序创造法则，落实为支撑体系与显相体系的协调构建。二者是"心物关系"在社会治理与修行实践上的完整展开，也是生命禅院"无为而无不为"社会理想的理论基础。
