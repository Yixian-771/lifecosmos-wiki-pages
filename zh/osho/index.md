# 奥修

**奥修**（Osho，1931—1990），印度思想家，在生命禅院体系中被定位为"知音"与"先驱"。导游雪峰高度认同奥修的反宗教组织、倡导真正自由的思想，并将其列为生命禅院配套读物之一；但雪峰明确指出，奥修缺乏生命观，不明白生命的原理和奥妙，"只来到了雪峰山下"，其静心法门不是究竟的、不是终极的——生命禅院的道路在奥修之上。
