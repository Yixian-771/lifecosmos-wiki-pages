# 人生观·价值观

**人生观是人对人生的目的、意义、价值的认识和观点；价值观是人对事物重要性的一种认识和评判及肯定和否定。**生命禅院认为，没有方向的人生是盲目的，没有价值观的人生是迷茫的——大多数人因持有错误的人生观与价值观而走错了人生方向，饱受苦难。真正符合上帝之道的人生观，以开心、快乐、自由、幸福为目的，以成仙进天国为归宿；普世价值观的首要一条，是信仰上帝。
