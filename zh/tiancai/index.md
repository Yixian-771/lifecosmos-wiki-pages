# 天才

**天才，是生命在前世前前世积累的意识素质于今生的显现，是全息宇宙赋予每个生命的潜在可能**——只要专注于某一方面、具备合适的土壤与自由，人人皆可成为天才；而杂务、应酬、兴趣广、不自由是毁灭天才的四大元凶。在生命禅院体系中，能成仙成佛者必定是天才，第二家园的生活模式是使天才得以充分彰显的根本出路。
