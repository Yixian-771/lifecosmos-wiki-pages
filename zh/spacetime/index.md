# 时空

**时空**，即时间与空间，是生命禅院宇宙观的基础结构坐标。时间是反物质，是物质运动状态的"记录器"；空间共有36维，连接20个平行世界。生命所处的时空层次，直接决定其存在形态与最终归宿。认识时空、超越时空，是修行觉悟的核心之一。
