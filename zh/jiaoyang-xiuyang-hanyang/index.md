# 教养·修养·涵养

教养，是从家庭、学校、社会中习得的文明礼貌；修养，是基于理论、知识、艺术、思想积累所形成的文明待人处事态度；涵养，是基于深厚道德素养而成就的控制情绪的能力。三者层层递进，缺一不可——缺教养者被厌恶，缺修养者被鄙视，缺涵养者被敬而远之。在生命禅院体系中，信仰决定三养，三养决定内心世界，内心世界决定言行，言行决定人生结果；三养既是人生必修课，也是通往天国的24个硬指标之一。
