# 情债

**情债**，是生命禅院修行体系中"五种债务"之一，指在人际往来与男女情爱中，获得了他人生理、心理或精神层面的帮助与慰藉，却未能对应回馈所欠下的债务。情债分人情债与情爱债两类，情缘未了亦成情债。情债不偿，必入轮回；债务清净，方得自由，方能向天国进发。
