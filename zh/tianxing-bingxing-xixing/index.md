# 天性·秉性·习性

**天性、秉性（禀性）、习性**是生命禅院体系对"性"的三层分类：天性是一类存在共同具备的道性，澄澈清明，体现上帝神佛的特征；秉性是某一类别的特性，即太极阴阳之德性；习性是单个个体附和生存环境所形成的惯性，即后天积累的理性。修行修炼的方向是：克服习性，摒弃秉性，恢复天性——与道合一，回归如来本性。
