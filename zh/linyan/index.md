# 灵眼

**灵眼**是生命禅院修行修炼体系中关于生命感知能力的核心概念，指人感知和观察**反物质世界**的内在能力——与感知物质世界的**肉眼**相对。

雪峰的核心定位：

> **肉眼看物质世界，灵眼看反物质世界。**
>
> "如果我们看不到物质世界的一切，证明我们的肉眼出了问题，瞎了，或闭了，或被蒙了；如果我们看不到反物质世界的一切，说明我们的灵眼出了问题，或是瞎了，或是闭了，或是被蒙了。"

灵眼的本质是**灵觉**（第六知觉系统）的感知器官——人有六大知觉系统（视觉、听觉、嗅觉、味觉、触觉、**灵觉**），灵眼是灵觉"观看"的通道。

灵眼**人人天生具备**，不需要额外添加，但受到贪欲、嗔恨、傲慢、嫉妒等"尘埃"的蒙蔽，灵觉便变得迟钝。灵觉越灵敏，人就越趋近仙人；灵觉越迟钝，人就越趋向浑人。

**开启灵眼**，是修行修炼走向高层生命空间的主要途径。
